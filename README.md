## 🎯 痛点洞察

### 大背景

- **AI 裁员潮**：2026 年各大公司大规模应用 AI，导致大量岗位被替代
- **经济压力**：原本有经济能力出行旅游的人，因失业/降薪导致资金紧张
- **旅行需求仍在**：工作压力大，更需要旅行放松，但预算有限

### 核心痛点

```
❌ 想旅行但资金不足
❌ 知道可以边旅行边创收，但不知道有哪些方式
❌ 信息分散在各大平台（活动发布、招聘、代购群），难以系统规划
❌ 不知道某个创收方式能赚多少钱、是否值得去
❌ 缺乏执行路径，不知道从哪开始
```

### 用户心声

> "我想出去走走，但刚被裁员，存款不多。有没有什么方式可以边旅行边赚点钱，至少覆盖旅费？"
> 
> "我知道有人去演唱会卖周边赚钱，但不知道具体怎么操作、能赚多少、风险是什么。"
> 
> "我会摄影，能不能旅行时帮人拍照赚旅费？但不知道哪里有机会、怎么获客。"

## 💡 核心价值

### 产品定位

**旅行创收助手** = 机会发现 + 收益测算 + 执行计划

帮助用户发现「边玩边赚钱」的旅行方案，从机会发现到执行落地全流程支持。

### 核心功能

| 功能模块 | 说明 | 用户价值 |
|:---|:---|:---|
| **机会发现** | 扫描全网活动/赛事/展会/热点，识别创收机会 | 不用自己到处找 |
| **玩法库** | 100 种旅行创收玩法，覆盖各技能水平 | 总有一款适合你 |
| **收益测算** | 基于历史数据预测收入，计算净收益 | 知道能赚多少钱 |
| **个性化匹配** | 技能×预算×时间 三维匹配推荐 | 推荐最适合的方案 |
| **执行计划** | 生成倒计时任务清单，一步步跟着做 | 知道从哪开始 |
| **获客支持** | AI 生成文案/海报/话术，帮助获客 | 降低获客门槛 |
| **风险提示** | 合规建议/竞争分析/备选方案 | 降低决策风险 |

## 🚀 功能特性

### 1. 智能推荐引擎

根据用户的**技能**、**预算**、**时间**三个维度，智能匹配最适合的创收方案。

```
输入：
- 技能：摄影、视频剪辑
- 预算：5000 元
- 时间：7 天
- 偏好：不想太累

输出：
1. 武汉樱花季旅拍（净收益 +5900 元）⭐⭐⭐⭐⭐
2. 厦门鼓浪屿旅拍（净收益 +4200 元）⭐⭐⭐⭐
3. 洛阳牡丹节摄影（净收益 +3800 元）⭐⭐⭐⭐
```

### 2. 收益测算模型

基于历史数据和市场调研，提供详细的收益预测。

```
收入预测：
- 服务单价：700 元/单
- 预计单量：12 单
- 总收入：8400 元

成本明细：
- 交通：800 元
- 住宿：1800 元
- 餐饮：1000 元
- 门票：120 元
- 总成本：2820 元

净收益：5580 元
投资回报率：198%
回本周期：3 单
```

### 3. 执行计划生成

自动生成倒计时任务清单，一步步跟着做。

```
D-14（3 月 18 日）：小红书发布旅拍预告
D-7（3 月 25 日）：接受预约，收取定金
D-3（3 月 29 日）：踩点拍摄位置
D-Day（4 月 1-7 日）：执行拍摄
D+3（4 月 10 日）：交付照片，收尾款
```

### 4. AI 获客支持

自动生成获客文案、海报、话术。

```
AI 生成内容：
- 小红书文案 × 5 篇
- 朋友圈文案 × 3 篇
- 宣传海报 × 2 款
- 客服话术 × 10 条
- 议价话术 × 5 条
```

## 📋 使用示例

### 示例 1：活动套利

**用户输入**：
```
我 4 月份有 3 天时间，预算 2000 元，想找个能赚钱的旅行方案，
听说演唱会卖周边能赚钱，具体怎么操作？
```

**AI 输出**：
```
【推荐方案】周杰伦演唱会周边售卖（净收益 +1800 元）

📅 基本信息
- 活动：周杰伦「嘉年华」演唱会 - 上海站
- 时间：4 月 12 日 -4 月 14 日（3 天）
- 地点：上海梅赛德斯 - 奔驰文化中心
- 预计人流：每场 1.8 万人，共 3 场

💰 收益测算
- 预期收入：周边售卖 50 单×80 元 = 4000 元
- 总成本：2900 元
- 净收益：+1100 元

📋 执行步骤
1. D-7：1688 采购周边商品
2. D-3：小红书发布售卖预告
3. D-1：出发上海，踩点售卖位置
4. D-Day：现场售卖，微信收款
5. D+1：复盘收益，发布经验分享

⚠️ 风险提示
- 合规风险：场馆周边可能禁止售卖
- 竞争分析：预计 10-15 个同行
- 压货风险：未售完可闲鱼转卖
```

### 示例 2：技能变现

**用户输入**：
```
我会摄影，4 月份有 7 天时间，预算 5000 元，
想找个能覆盖旅费甚至赚钱的旅行方案
```

**AI 输出**：
```
【推荐方案】武汉樱花季旅拍（净收益 +5900 元）

📅 基本信息
- 目的地：湖北武汉
- 时间：4 月 1 日 -4 月 7 日（7 天）
- 最佳拍摄点：东湖樱园、武汉大学、汉口江滩

💰 收益测算
- 预期收入：旅拍服务 12 单×700 元 = 8400 元
- 总成本：2820 元
- 净收益：+5580 元

📋 执行步骤
1. D-14：小红书发布旅拍预告
2. D-7：接受预约，收取 30% 定金
3. D-3：踩点拍摄位置，规划路线
4. D-Day：执行拍摄，每天 2-3 单
5. D+3：交付精修照片，收尾款
```

### 数据模型

```typescript
interface TravelOpportunity {
  id: string;
  name: string;
  category: string;           // 6 大类别
  description: string;
  skills: string[];           // 所需技能
  budget: {
    min: number;
    max: number;
  };
  duration: {
    min: number;              // 最少天数
    max: number;              // 最多天数
  };
  income: {
    min: number;              // 最低收益
    max: number;              // 最高收益
    currency: string;
  };
  destinations: string[];     // 适合目的地
  seasons: string[];          // 适合季节
  difficulty: 'easy' | 'medium' | 'hard';
  riskLevel: 'low' | 'medium' | 'high';
  steps: Step[];
  tips: string[];
  risks: string[];
}
```
